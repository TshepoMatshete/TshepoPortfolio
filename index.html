Latest Project

The latest project I’m working on is a web application that lets people enjoy the convenience of a laundromat from the comfort of their homes by ordering laundry pickup and delivery services. I built the project using Next.js, with TypeScript for the backend, React for the frontend, and Supabase for the database and authentication. I also integrated AI to help customers choose the best service plan based on the type and amount of clothes they want dry cleaned. The project is called GoWash "when you gowish... we GoWash for you" :)
