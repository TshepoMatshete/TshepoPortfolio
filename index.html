About me

I’m a final year computer science student at the university of the witwatersrand who enjoys coding, problem-solving, and learning new technologies. I like building things, experimenting with ideas,critical thinking, and improving my skills through hands-on projects.

I’m particularly interested in growing into a strong software engineer by working on real systems(like my GoWash project) that people can actually use. I like taking ideas from scratch, designing how the pieces should work, and then building, testing, and improving them until they’re solid. Through my journey of developing my web application, I’ve learned that I enjoy writing clean code, solving tricky implementation problems, and continuously refining solutions to be faster, simpler, and more reliable.

Latest Project

The latest project I’m working on is a web application that lets people enjoy the convenience of a laundromat from the comfort of their homes by ordering laundry pickup and delivery services. I built the project using Next.js, with TypeScript for the backend, React for the frontend, and Supabase for the database and authentication. I also integrated AI to help customers choose the best service plan based on the type and amount of clothes they want dry cleaned. The project is called GoWash "when you gowish... we GoWash for you" :). Through building it, I’ve been getting more comfortable thinking about scalability, user experience, and how different parts of a production system communicate with each other using the scrum. It’s been a great way to practice writing maintainable code while solving real problems that a business and its customers might face.
